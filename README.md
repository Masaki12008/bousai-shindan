# 🛡️ 防災キット診断アプリ

**あなたの家族に本当に必要な防災グッズがAIで診断できる無料Webアプリです。**

👉 [アプリを使ってみる](#) <!-- デプロイ後にURLを入れてください -->

---

## 📱 機能

- **4ステップの簡単診断** — 家族構成・住環境・備蓄状況・避難スタイルを入力するだけ
- **AIパーソナライズアドバイス** — Claude AIがあなたの家族に合ったアドバイスを生成
- **優先度つき不足リスト** — 最重要・高・中の3段階で必要なものを整理
- **楽天・Amazon直接リンク** — 必要なアイテムをそのまま購入できる
- **防災豆知識** — 役立つ豆知識を自動ローテーションで表示
- **すでに持っているものを除外** — 差分だけ表示するので無駄なし

---

## 🎯 こんな人に

- 防災グッズを揃えたいけど何から買えばいいかわからない
- 家族構成に合った備蓄を知りたい
- 乳幼児・高齢者・ペットがいる家庭
- 洪水・地震・津波など地域リスクに合わせて備えたい

## 🛠️ 技術スタック

- **フロントエンド**: React + Vite
- **AI**: Anthropic Claude API
- **デプロイ**: Vercel
- **収益化**: 楽天アフィリエイト・Amazonアソシエイト

## 🚀 ローカルで動かす

```bash
git clone https://github.com/あなたのID/bousai-diagnostic.git
cd bousai-diagnostic
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

---

## 📦 デプロイ

```bash
npm run build
```

Vercelにリポジトリを接続すると自動デプロイされます。
